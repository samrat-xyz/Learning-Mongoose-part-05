# Getting Started with Mongoose (Lesson 04)


A simple REST API built with **Node.js, Express, and Mongoose** to perform basic CRUD (Create, Read, Update, Delete) operations on users.

---

##  Features

- Create a new user
- Get all users
- Update a user by ID
- Delete a user by ID
- MongoDB database using Mongoose
- RESTful API structure

---

##  Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
